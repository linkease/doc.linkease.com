# 套餐问题排查

> 解决套餐购买、绑定、续费等问题

---

## 套餐不显示

### 症状
- 已购买套餐但控制台不显示
- 套餐列表为空

### 可能原因

#### 1. 套餐已过期
过期的套餐不会继续显示在列表中。

**解决：**
- 购买新套餐
- 或在过期前续费

#### 2. 购买了易有云套餐
易有云和 DDNSTO 是两个独立的产品，套餐不通用。

**解决：**
- 确认购买的是 DDNSTO 套餐
- 如买错，联系客服处理

## 套餐绑定问题

### 症状
- 无法绑定套餐到设备
- 提示套餐已被绑定

### 解决

#### 绑定新设备

1. 登录 DDNSTO 控制台
2. 设备管理 → 找到旧设备 → 套餐管理 → 「解绑」

![解除绑定](../resources/question/question2.jpg)

![解除绑定](../resources/question/question2.png)

3. 新设备 → 套餐管理 → 「+套餐管理」 → 找到“未使用”的套餐 → “绑定至本机”

![绑定套餐](../resources/question/question6.jpg)

![绑定套餐](../resources/question/question6.png)

## 切换服务器

### 操作步骤

1. 登录 DDNSTO 控制台
2. 设备管理 → 找到设备 → 服务器选择
3. 选择目标服务器

![切换服务器](../resources/question/question3.jpg)

4. 服务器切换成功后，外网域名的域名地址也会变更

![选择服务器](../resources/question/question4.jpg)

## 套餐续费

### 续费流程

#### 套餐未过期

1. 设备管理 → 点击设备 → 套餐管理 → 「续费」

![续费](../resources/question/xf7.jpg)

2. 选择适合自己的续费套餐，完成支付即可

![续费](../resources/question/xg7.png)

#### 套餐已过期

1. 设备管理 → 点击设备 → 套餐管理 → 「续费」

![购买新套餐](../resources/question/xf2.jpg)

2. 选择适合自己的续费套餐，完成支付即可

![购买新套餐](../resources/question/xf3.jpg)

## 常见问题

### Q: 已购买套餐的设备不小心删除了怎么办？

A: 即使删除了设备，套餐依然还在。重新添加设备后就可以绑定已购买套餐。

### Q: 套餐可以退款吗？

A: 虚拟商品一经售出，原则上不支持退款。如有特殊情况，请联系客服。

### Q: 套餐可以转让吗？

A: 不可以，套餐绑定到账号，不支持转让。

### Q: 免费套餐有什么限制？

A: 免费套餐通常有：
- 带宽限制（如 1Mbps）
- 域名数量限制（如 1 个）
- 功能限制（如不支持远程应用）
