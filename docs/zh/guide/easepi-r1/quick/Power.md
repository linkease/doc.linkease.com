* EasePi-R1 通电自启动！

![image](./image/power.png)

```
重置：长按按钮 5秒 左右，PWR 指示灯闪时松开恢复出厂设置
强制断电：长按超过 10 秒强制断电
```

* 22.03 版本及 24.10.x-20250704之前的固件版本：
```
关机：单击（短按）按钮
```

* 24.10.2-20250704 以后的固件版本：
```
关机：连续短按两次按钮
```
