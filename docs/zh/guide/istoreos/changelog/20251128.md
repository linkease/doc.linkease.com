### 系统
1. 固件集成 kmod-ipt-ipopt 模块，以避免从 22.03 保留配置升级以后 mwan3 失效问题。注意 mwan3 可能还是需要重新安装。
2. IPv6 地址管理服务（odhcpd）支持将无状态分配记录保存到文件中，即使系统重启也能继续维护分配记录。新刷机或恢复出厂设置会默认开启，对于存量设备，执行`uci set dhcp.odhcpd.piofolder=/tmp/odhcpd-piofolder; uci commit dhcp; /etc/init.d/odhcpd restart`即可开启。

### 设备
* Rockchip 系列：集成 r8125 和 r8168 可选驱动，在 系统-杂项设置-驱动 中可管理。注意，如果之前在软件包中安装过 kmod-r8125 或 kmod-r8168 ，或者卸载过 kmod-r8169 ，那么升级前先“修复系统软件”（从 iStore 中安装“系统便利工具”1.0.17或以上版本），以免驱动不按预期加载导致后台都进不去（至少要有 kmod-r8169 兜底）。（r8125 驱动支持 2.5G 网卡，r8168 驱动支持 Nanopi R4S 或 RK35xx 系列上的 PCIE 千兆网卡）
* Radxa E24C：新增此机型。
* Radxa E54C：修复 USB 3.0 接口只跑到 2.0 速度问题；修复 RTC 偶尔初始化失败问题。
