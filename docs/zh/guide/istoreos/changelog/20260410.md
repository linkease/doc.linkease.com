### 系统
* 系统升级/在线升级：在线升级过程增加手动刷新页面提示，并推迟5秒检查系统重新上线；一次升级中，检查重新上线的标志文件时使用相同URL，解决部分电脑多网关或网关不是被升级机器环境下，浏览器检查标志文件不断超时问题

### 应用
* OAF（应用过滤）：修复 6.1.6 后端服务 oafd 严重死循环嵌套内存泄漏 bug，一旦触发，系统会在几秒内耗尽内存并 OOM（非常见的缓慢泄漏），表现就是前端卡顿或卡死，一段时间后恢复，可能部分系统服务会被重启。bug 触发部分条件是安装了特定版本的特征库，并在用户列表查看详情。注意上游 6.1.7 已修复此 bug。
* LuCI：backport 上游 WiFi 扫描界面高风险 XSS 漏洞修复。通过特意构造的 SSID，此漏洞可在 WiFi 扫描时注入 HTML/JS 代码。

### 设备
* EasePi R2：新增支持

### 风险提示
* 部分用户反馈，自 20260320 版本开始，R4S/T68M 启动失败或网卡初始化失败问题。
  * 如果已经升级 20260320 及之后版本未遇到此问题，那继续升级应该也没问题；
  * 如果已经遇到此问题，回退到 20251231，等开发拿到日志或测试机并解决问题以后再升级；
  * 如果还未升级，可等待问题解决，或做好备份以后再尝试升级。
