### 小版本升级
OpenWRT 版本从 24.10.5 升级到 24.10.6。

OpenWRT 24.10.6 发布日志在此查看 https://github.com/openwrt/openwrt/releases/tag/v24.10.6

### 系统
1. 修复 RTL8367RB 交换机（芯片）保存配置以后“交换机”页面加载超时问题（受影响机型 Radxa E24C/E54C）。如果之前已经触发，升级以后执行这个命令修复`uci delete 'network.@switch[0].enable_vlan4k' ; uci commit network; /etc/init.d/network reload` 

### 应用
* 统一文件共享：修复 WEBDAV 服务在某些 APP 里上传失败问题 

### 设备
* LYT T68M：修复 PCIE 及相应网卡概率性初始化失败问题
* Radxa E54C：修复 “交换机” 页面显示的 LAN 口标签与外壳不一致问题（顺序相反），仅对重新刷机或恢复出厂设置生效

### 风险提示
* 部分用户反馈，自 20260320 版本开始，R4S 启动失败或网卡初始化失败问题。
  * 如果已经升级 20260320 及之后版本未遇到此问题，那继续升级应该也没问题；
  * 如果已经遇到此问题，回退到 20251231，等开发拿到日志或测试机并解决问题以后再升级；
  * 如果还未升级，可等待问题解决，或做好备份以后再尝试升级。
