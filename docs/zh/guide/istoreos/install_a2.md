### EasePi A2

[EasePi](/zh/guide/easepi-r1/easepi.html) 是由易有云团队专为家庭智能网关、轻 NAS 设计的硬件产品系列。 
`EasePi-A2`是高性能单网口多功能小主机，支持 Android 系统与 Docker 环境，也支持 iStoreOS/armbian 等系统，并提供双 NVMe 插槽以满足多种应用场景。


#### 配置详情：

| 项目 | 参数 | 说明 |
|---|---|---|
| 处理器 | RockChip RK3568 | 四核 Cortex-A55 2.0GHz |
| 内存 | 4GB | LPDDR4 |
| EMMC | 32GB | — |
| ETH0 | RTL8125BG | PCIe 接口，2.5GbE |
| 红外 | 标准红外接收器 | 前置 |
| OLED 显示屏 | 128×64 OLED 单色显示屏 | I2C 3.3V |
| NVMe | PCIe to NVMe ×2 | 2 条全尺寸支持 |
| USB | USB 3.0 | 5Gbps |
| OTG | Type‑C | 仅提供 OTG 服务 |
| HDMI | HDMI 2.0 | — |
| Key1 | Power Key | 开关机键 |
| Key2 | Mask ROM | 固件升级 |
| LED | POW LED / RUN LED | 默认双绿色，可选其他颜色 |
| Wi‑Fi | SDIO AP6255 | 双频 |
| FAN POWER | 风扇接口 ×2 | GPIO 控制 |
| Debug TTL | 调试接口 | 板载调试接口 |
| 电源接口 | DC12V2A | — |
| 外壳尺寸 | 135×115×28mm | 钣金折弯金属外壳 + 静电喷塑工艺 |
