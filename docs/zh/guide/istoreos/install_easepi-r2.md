EasePi R2 如何进入 Maskrom 模式：

* 数据线一端连接电脑，另一端 C 口接入 R2 的 C 口(OTG)；

![install.png](./install/easepi/r2-2.png)

* 靠近 KEY 为 ROM 键，持续按住 ROM 键，机器进入 Maskrom 模式（电脑刷机工具发现 Maskrom 设备后，松开！）。

![install.png](./install/easepi/r2-1.png)
